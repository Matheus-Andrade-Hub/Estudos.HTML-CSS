Importando fontes.

Fontes importadas:

Para importar esta fonte eu fui no google fontes, escolhi uma fonte que fosse do meu gosto e importei ela, depois disso eu adicionei ela usando o seletor (font) no css.

Desta maneira:

h1 { font: italic bolder 2em "Sriracha", "arial", "italic", sans-serif }

h2 { font: italic bolder 27px "Gesso", arial, serif }

font: o seletor font e um jeito mais simples de organizar as fontes, voce organiza na seguinte ordem, font-style -> font-weight -> font-size -> font-family.

Textos.

Textos alinhados:

para alinhas os textos eu usei o seletor text-align e text-ident.

Desta maneira:

p { text-align: left; text-align: justify; text-indent: 15px; }

text-align: serve para alinhar o texto.
text-indent: serve para dar margem no inicio do paragrafo.
