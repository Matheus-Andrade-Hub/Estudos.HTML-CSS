Caixa box-level.

Caixas do tipo box-level ocupam toda a largura do paragrafo se essa largura nao for definida, alem de que se voce colocar outro elemento ou conteudo, ele sera quebrado para a linha de baixo, voce pode alterar uma caixa para inline-level utilizando (display: inline;) com css e tambem alterar para box-level utilizando (display: block;).
