Classes e Pseudo Elementos.

ids (#) :

O id e um identificador unico para selecionar um elemento especifico do html, usamos (#) para manipular elemetnos com id no css.

class (.) :

class tambem e um identificador que identifica elementos html,usamos (.) para manipular elementos que sejam da mesma classe, porem todos os elementos que tiverem class, serao identificados como um so, exemplo: se usar um color: white; em um elemento que se chame botao, todos os elementos que se chamem botao vao ficar com a mesma cor

pseudo classes (:) :

As pseudo classes sao usadas para definir um estado especifico de um elemento, um exemplo uma pseudo classe (div:hover (Quando o cursor estiver em cima)), usamos (:) para utilizar as pseudo classes.

pesudo elementos (::) :

Os pesudo elementos sao usado para criar ou estilizar partes especificas de um elemento, Usamos (::) para criar um pesudo elemento, exemplo (div::before (insere conteudos depois do elemento))
